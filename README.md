# eops-web

## 项目介绍
eops前端源码。

## 软件架构
node+vue+element


## 安装教程

* 安装node
* 安装vue
* 安装element
* 安装依赖npm install
* 修改config/index.js，build路径
* 启动debug服务npm run dev
* 访问localhost:80

## 使用说明
### 前后台联调
* 后台端口10101
* 文件服务端口10100
* 修改config/index.js，代理url端口
* 启动后台服务
* 启动前台debug服务
* 访问localhost
